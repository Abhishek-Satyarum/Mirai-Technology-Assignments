# The Memory Vault – Stateful AI Chatbot

## ✨ Features

- Stateful conversation using `st.session_state`
- Remembers previous messages during the session
- Multiple AI personalities
- Dynamic roleplay using prompt engineering
- Adjustable personality intensity
- Modern chat interface with `st.chat_message()`
- Native chat input using `st.chat_input()`
- Dynamic emoji avatars for different personalities
- Powered by Google Gemini 2.5 Flash
